如何高概率的胡清一色

Feb 28, 2017 | Hits

中华文化博大精深，麻将这个东西，实在是太美妙了。然而水平有限，总是输多赢少，赢得时候多半也是屁胡，别人胡的时候经常是清一色啊，混一色啊什么的，把我赢得一秒回到解放前Orz所以经过查阅多份资料，总结了几条胡清一色的Tips……

什么是清一色

首先，我们需要知道什么是清一色。在麻将中，所有的牌分为两大类，字牌和色牌，字牌包括东南西北四种风，白班，红中以及发财，色牌则包括万，条，筒三种。
所以，顾名思义，清一色简单的来说就是胡牌时牌型里有且只有一种色牌，与此相对的，还有一种牌型叫做混一色，指的是牌型中只有字牌和一种色牌混合，今天暂且不做讨论。

拿到牌后的分析

在分到牌后，我们应该做的第一件事是什么？通过观察古往今来的麻将典籍，前辈们总结出，应该首先计算手上的搭子数目。什么是搭子呢？搭子是指，再来一张牌就可以组成一副的牌，比如对子，再比如三万和五万。
如果手中的牌低于四个搭子，强烈建议采用守势，可以先把边边的牌先丢掉一部分，比如一万二万。
那什么是比较好的搭子嘞？比如说二三三万，就是一个非常好的搭子，一万时边牌，很容易吃到，自己摸到一三四万也可以凑齐，还可以碰三万，必要的时候还可以打掉二万，把三万做将。

其他

其实吧，胡九种牌的牌型也有，但是那就比天胡还罕见列，所以我们关于清一色牌型的介绍就到此为止啦～
在查资料的过程中，我发现了一些麻将界的俗语（滑稽），或许会有一些迷信的意味，但是放在这里仅供参考。

  • 头不吃

    对方打出的头一张牌不吃碰。(为了观察自己第一轮的摸牌运气怎么样)

  • 扬黑抑红

    对局中，应该抑制胜者，帮助败者。

  • 牌运来时，趁胜追击

    牌运好时，应尽量胡牌，不要心软，否则气跑了，运也跑了。

  • 避重就轻

    自己已经没有胡牌希望，就应避开拿”大牌”者，而让拿”小牌”者胡牌或臭庄，这也算是作战方式之一。

结语

一开始的时候肯定记不住这些，所以要每次摸到牌都敏锐分析～祝我自己每局都能杠上开花自摸清一色！
