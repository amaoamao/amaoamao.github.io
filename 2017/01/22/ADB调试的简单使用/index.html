ADB调试的简单使用

Jan 22, 2017 | Hits

ADB，全称 Android Debug Bridge，即安卓调试桥，附带于Android SDK中，位于SDK目录的platform-tools文件夹下。
adb的基本命令如下

其中，-d代表device，意为当前唯一通过USB连接的设备，-e代表emulator，意为当前唯一运行的模拟器。当同时有多个设备或模拟器连接时，需要使用-s来通过serialNumber指定命令目标。

实际上，adb不仅可以通过USB连接设备，也可以通过局域网进行无线连接。
首先，连接USB

接下来就可以拔掉USB连接线了

PackageManager

使用pm可以查看已安装的应用

同时，可以使用grep命令来过滤

install & uninstall

adb install < apk文件 >
adb uninstall < 软件包名 >
adb uninstall -k < 软件包名 >
如果加-k参数，为卸载软件，但是保留配置和缓存文件

push & pull

adb push <本地路径> <远程路径>
把PC的文件或者文件夹复制到设备上
adb pull <远程路径> <本地路径>
把设备上的文件或者文件夹复制到PC
